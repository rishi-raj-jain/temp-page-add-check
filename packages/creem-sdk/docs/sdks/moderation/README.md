# Moderation

## Overview

### Available Operations

* [screenPrompt](#screenprompt) - Screen a prompt

## screenPrompt

Evaluate a text prompt against content policies before generation. This endpoint is experimental and may change.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="screenPrompt" method="post" path="/v1/moderation/prompt" -->
```typescript
import { Creem } from "creem";

const creem = new Creem({
  apiKey: process.env["CREEM_API_KEY"] ?? "",
});

async function run() {
  const result = await creem.moderation.screenPrompt({
    prompt: "<value>",
  });

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { CreemCore } from "creem/core.js";
import { moderationScreenPrompt } from "creem/funcs/moderationScreenPrompt.js";

// Use `CreemCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.
const creem = new CreemCore({
  apiKey: process.env["CREEM_API_KEY"] ?? "",
});

async function run() {
  const res = await moderationScreenPrompt(creem, {
    prompt: "<value>",
  });
  if (res.ok) {
    const { value: result } = res;
    console.log(result);
  } else {
    console.log("moderationScreenPrompt failed:", res.error);
  }
}

run();
```
